# 🏦 Praveena's Bank: Smart Fraud Detection System

An AI-driven fraud detection system designed to analyze transaction patterns, handle imbalanced datasets, and identify suspicious banking activities in real-time.
An intelligent machine learning application developed to detect and prevent fraudulent banking transactions, ensuring secure financial operations. 

## 🚀 Project Overview
With financial fraud becoming increasingly sophisticated, this project focuses on building robust, data-driven security measures. **Praveena's Bank** utilizes advanced machine learning classification algorithms to analyze transaction patterns, manage highly skewed/imbalanced datasets, and flag anomalies before they impact users.

## 🛠️ Key Features
* **Smart Anomaly Detection:** Classifies legitimate vs. suspicious account activities with high precision.
* **Robust Data Pipeline:** Implements rigorous preprocessing, scaling, and feature engineering for skewed data.
* **Security-Centric Evaluation:** Optimized specifically to minimize false negatives, ensuring unauthorized transactions are caught.
